# Focus Timer

A small Pomodoro timer for getting work done without losing track of breaks. Built with plain HTML, CSS, and JavaScript. No frameworks, no build step, no dependencies.

## Features

- Focus, short break, and long break modes
- Session lengths you can adjust, saved between visits with localStorage
- Circular progress ring and a dot tracker for the four session cycle
- A long break suggested automatically after every four focus sessions
- Keyboard shortcuts (space to start or pause, R to reset)
- An optional chime and a desktop notification when a session ends
- Works offline and adapts to small screens

## Running it

Open `index.html` in any modern browser. That is all it needs.

## Why I built it

I wanted a timer that was quick to load, kept its settings, and did not nag me with ads or accounts. It started as a personal tool and turned into a clean little project worth sharing.
